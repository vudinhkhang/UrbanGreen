Nhập Danh Sách Cây

Hướng Dẫn Nhập File

Định dạng file hỗ trợ: .xlsx, .csv, .xls

Cột bắt buộc:

  • Mã cây - Mã định danh của cây (phải là duy nhất)
  • Loài - Tên loài cây (phải tồn tại trong hệ thống)
  • Chiều cao (m) - Chiều cao của cây (số thực)
  • Trạng thái - Trạng thái cây:
    • Tốt (TOT)
    • Sâu bệnh (SAU_BENH)
    • Nguy hiểm (NGUY_HIEM)
  • Vĩ độ - Tọa độ vĩ độ (latitude, số thực)
  • Kinh độ - Tọa độ kinh độ (longitude, số thực)

Cột tùy chọn:

  • Địa chỉ - Địa chỉ cây
  • Đường kính thân (cm) - Đường kính thân cây
  • Diện tích vòm (m²) - Diện tích vòm che phủ
  • Năm trồng - Năm cây được trồng

Mẹo: Bạn có thể tải file mẫu bằng cách xuất dữ liệu hiện tại rồi sửa lại.
